# SMART Innovation - 로드맵 타임라인 대시보드 PRD

## 1. 개요

### 1.1 프로젝트 목적
과제별 로드맵 일정을 웹 대시보드에서 직접 입력하고, 이미지와 같은 Gantt 스타일 타임라인으로 자동 시각화하는 시스템.

### 1.2 배경
기존에는 PPT/Excel로 수동 작성하던 로드맵 타임라인을 웹 대시보드화하여 실시간 등록·수정·공유가 가능하도록 개선.

### 1.3 기술 스택
| 항목 | 내용 |
|------|------|
| Framework | Next.js 16 (App Router) |
| Language | TypeScript |
| Styling | Tailwind CSS v4 |
| Runtime | Node.js |
| Port | 4000 |

## 2. 기능 요구사항

### 2.1 과제 입력 폼

| 필드 | 설명 | 형식 |
|------|------|------|
| Level (과제 그룹) | 과제의 상위 카테고리 (예: MES, CMMS, NIR) | 기존 그룹 선택 또는 신규 그룹 생성 |
| 과제명 | 개별 과제 이름 | 텍스트 |
| 시작일 | 과제 시작 월 | YYYY-MM |
| 종료일 | 과제 종료 월 | YYYY-MM |
| 담당자 | 과제 담당자명 | 텍스트 (선택) |
| 상태 | 진행 상태 | 예정 / 진행중 / 완료 / 핵심 마일스톤 |

- 신규 그룹명 입력 시 자동으로 카테고리 생성
- 필수 필드(그룹, 과제명, 시작일, 종료일) 미입력 시 제출 불가

### 2.2 타임라인 시각화

#### 헤더 영역
- 최상단: 연도 단위 컬럼 그룹 표시 (예: 2026년, 2027년, 2028년)
- 그 아래: 월 단위 컬럼 표시 (1~12월)
- 연도가 바뀌는 지점에 굵은 구분선 표시

#### 행 구성
- 좌측 고정 라벨: 카테고리(Level) 명칭 (색상 뱃지)
- 우측 차트 영역: 과제별 시작점(●) ~ 종료점(●)을 선으로 연결
- 과제명: 바 상단에 표시
- 담당자명: 바 우측 끝에 표시

#### 날짜 범위 자동 조정
- 등록된 과제의 가장 이른 시작월 ~ 가장 늦은 종료월 기준으로 차트 범위 자동 설정
- 앞뒤 1개월 여유 자동 추가

#### 상태별 색상 코드
| 상태 | 색상 |
|------|------|
| 예정 (planned) | 회색 (#9ca3af) |
| 진행중 (in-progress) | 녹색 (#00733C) |
| 완료 (completed) | 녹색 (#00733C) |
| 핵심 마일스톤 (critical) | 빨간색 (#dc2626) |

#### 카테고리별 색상
카테고리(Level)별로 자동 배정되는 고유 색상 (최대 10가지 순환):
`#00733C`, `#2563eb`, `#7c3aed`, `#dc2626`, `#ea580c`, `#0891b2`, `#4f46e5`, `#be123c`, `#15803d`, `#b45309`

### 2.3 과제 목록 패널

- 등록된 전체 과제를 카테고리별 그룹으로 표시
- 총 과제 수 표시
- 행 호버 시 삭제(✕) 버튼 노출
- 스크롤 가능 (최대 높이 400px)

### 2.4 유틸리티 기능

| 기능 | 설명 |
|------|------|
| 샘플 데이터 로드 | MES / CMMS / NIR 기반 예시 데이터 자동 로드 |
| 전체 초기화 | 등록된 모든 과제 및 카테고리 삭제 |

## 5. 데이터 모델

### TimelineTask
```typescript
interface TimelineTask {
  id: string;               // 고유 ID (랜덤 생성)
  category: string;         // 과제 그룹 (Level 1)
  name: string;             // 과제명
  startDate: string;        // 시작월 (YYYY-MM)
  endDate: string;          // 종료월 (YYYY-MM)
  assignee: string;         // 담당자
  color: string;            // 카테고리 색상
  isMilestone?: boolean;    // 마일스톤 여부
  status?: "planned" | "in-progress" | "completed" | "critical";
}
```
