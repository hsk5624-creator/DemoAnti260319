# 제조경영 예산 대시보드 PRD

## 1. 개요

제조부문 예산 담당자가 **매주 다운로드한 실적 XLSX 파일을 업로드**하면,
계획 대비 실적·집행률·가용예산을 부서/계정별로 즉시 확인할 수 있는 웹 대시보드.

---

## 2. 입력 데이터

### 2-1. 주간 실적 파일 (매주 업로드)
- 형식: `예산 sample(26.02).XLSX` 패턴
- 시트: 1개 (sheet1)
- 행 수: ~545행 (부서 × 계정 × WBS 조합)

### 2-2. 연간 사업계획 파일 (연초 1회 업로드)
- 형식: `예산 sample(연간 사업계획).XLSX` 패턴
- 컬럼: 예산부서/계정/WBS 분류 + 예산합계/실적합계/잔액합계 + **월별(1~12월) 예산·실적·잔액**
- 행 수: ~530행

### 3-2. 집계 규칙
- **팀 단위**만 집계 대상. 담당/본부/부문 레벨(임원예산)은 **제외**
- 제외 부서 코드: `120000(제조부문)`, `120001(품질본부)`, `120002(품질경영담당)`, `120007(품질관리담당)`, `120013(진천품질담당)`, `120018(케미컬생산담당)`, `120030(바이오생산담당)`, `120040(진천생산담당)`, `120050(생산기술담당)`, `120055(운영지원본부)`, `120057(생산지원담당)`

## 4. 핵심 지표 (KPI)

| 지표명 | 계산식 | 기준 |
|--------|--------|------|
| **연간 집행률** | 총 실적(Y) ÷ 총 예산(Y) × 100 | 연간 |
| **월간 집행률** | 실적합계(M) ÷ 총 예산(M) × 100 | 당월 |
| **계획 대비 차이(Y)** | 총 실적(Y) − 계획 예산(Y) | 연간 |
| **계획 대비 차이(M)** | 실적합계(M) − 계획 예산(M) | 당월 |
| **가용 예산(Y)** | 총 예산(Y) − 총 실적(Y) | 연간 |
| **예산 잔액(M)** | 총 예산(M) − 실적합계(M) | 당월 |

## 5. 화면 구성

### 5-1. 상단 요약 카드 (전체 합계, 임원예산 제외)
- 연간 총 예산 / 연간 누적 실적 / 연간 집행률
- 당월 총 예산 / 당월 실적 / 당월 집행률
- 가용 예산(Y) / 예산 잔액(M)

### 5-2. 조직별 집행 현황 테이블

**기본 뷰 — 담당 단위 집계**
- 기본적으로 담당(중간 관리) 레벨로 묶어서 표시
- 본부직속 팀(데이터보증팀, 운영지원팀)은 별도 행으로 표시
- 컬럼: 부서명 | 연간 예산 | 연간 실적 | 집행률(%) | 계획 대비 차이 | 가용 예산 | 당월 예산 | 당월 실적 | 당월 집행률

**드릴다운 — 팀 단위 펼치기**
- 담당 행 클릭(또는 ▶ 아이콘 클릭) → 해당 담당 소속 팀 목록이 인라인으로 펼쳐짐
- 다시 클릭하면 접힘 (accordion 방식)
- 팀 행은 들여쓰기로 계층 표현
- 여러 담당을 동시에 펼칠 수 있음

**색상 기준 (집행률)**
- 🔴 90% 초과: 예산 소진 위험
- 🟡 70~90%: 주의
- 🟢 70% 미만: 정상

```
[ 조직별 집행 현황 ]

▶ 품질경영담당          | 35,200,000 | 1,500,000 | 4.3% | -500,000 | 33,700,000 | ...
  └ 컴플라이언스팀       |  5,000,000 |   300,000 | 6.0% | ...
  └ 품질보증1팀          | 18,000,000 |   800,000 | 4.4% | ...
  └ 품질보증2팀          | 12,200,000 |   400,000 | 3.3% | ...

▶ 품질관리담당          | 80,000,000 | ...
  (접힌 상태)

  운영지원팀 (본부직속)  |  8,000,000 | ...
```

### 5-3. 계정별 집행 현황

**기본 뷰 — 계정별 집행률 차트**
- 가로 바 차트, Y축: 계정명, X축: 금액 또는 집행률
- 각 바에 집행률(%) 레이블 표시
- 집행률 높은 순 정렬
- 계정 42종을 상위 카테고리로 그룹핑하여 색상 구분
  - 복리후생비 / 소모품비 / 수선유지비 / 여비교통비 / 교육훈련비 / 기타

**상세 뷰 — 계정 클릭 시 (부서별 사용 현황)**
- 차트의 특정 계정 바 클릭 → 패널 또는 모달로 상세 표시
- 해당 계정을 집행한 부서 목록을 **집행 금액 내림차순** 정렬
- 표시: 부서명 | 연간 실적 | 집행률 | 미니 바 차트
- 상단에 "계정명 — 전체 집행 현황" 제목 표시

```
[ 소모품비-생산(MRO) 클릭 시 ]

  ┌─────────────────────────────────────────┐
  │ 소모품비-생산(MRO) — 부서별 집행 현황  │
  │ 전체: 85,000,000원 / 집행률 42.5%      │
  │                                         │
  │ 바이오생산1팀    ████████  32,000,000   │
  │ 바이오생산2팀    ██████    24,000,000   │
  │ 생산1팀          ████      15,000,000   │
  │ 생산2팀          ██         9,000,000   │
  │ ...                                     │
  └─────────────────────────────────────────┘
```

### 5-4. 월간 누적 계획 vs 실적 차트 (연간 사업계획 파일 기반)

**누적 집계 방식**
- 연간 사업계획 파일의 월별 예산·실적을 1월부터 해당 월까지 누적 합산
- 실적 파일 기준월까지만 실적 표시 (이후 월은 계획선만 표시)

| 월 | 월 계획 | 월 실적 | 누적 계획 | 누적 실적 |
|----|---------|---------|-----------|-----------|
| 1월 | 100만원 | 50만원 | 100만원 | 50만원 |
| 2월 | 100만원 | 60만원 | 200만원 | 110만원 |
| 3월 | 100만원 | 70만원 | 300만원 | 180만원 |
| 4월 | 100만원 | — | 400만원 | — |

**차트 구성**
- X축: 1~12월
- Y축: 누적 금액 (원)
- 라인 1 (파란색): 누적 계획 예산 — 12월까지 전체 표시
- 라인 2 (초록/빨강): 누적 실적 — 기준월까지만 표시
- 기준월 수직선으로 "현재" 위치 표시
- 각 월 포인트 hover 시 tooltip: 누적 계획 / 누적 실적 / 차이 / 누적 집행률

**누적 차이 요약 (차트 상단)**
- 기준월 기준: 누적 계획 XXX원 vs 누적 실적 XXX원
- 차이: ▲/▼ XXX원 (집행률 XX%)
- 계획 대비 초과/미달 여부 색상 표시

### 5-5. 파일 관리 영역

**연간 사업계획 파일 (상단 고정)**
- 최초 1회 업로드, localStorage에 유지 (새로고침 후에도 보존)
- 업로드된 파일명 표시, 재업로드 버튼 제공

**실적 파일 관리 (월별 누적)**
- 실적 XLSX 파일을 여러 개 업로드하여 목록으로 관리
- 파일명에서 월 정보 자동 파싱 (`예산 sample(26.02).XLSX` → `2026년 02월`)
- 업로드된 파일 목록을 **체크박스** 형태로 표시, 라디오 선택으로 조회 기준월 변경
- 선택된 파일 기준으로 전체 대시보드 데이터 갱신

```
┌──────────────────────────────────────────────┐
│ 연간 사업계획       예산 sample(연간 사업계획).XLSX  [변경] │
├──────────────────────────────────────────────┤
│ 실적 파일                          [+ 파일 추가] │
│                                              │
│  ○ 2026년 01월   예산 sample(26.01).XLSX      │
│  ● 2026년 02월   예산 sample(26.02).XLSX  ← 현재 선택  │
│  ○ 2026년 03월   예산 sample(26.03).XLSX      │
│                                              │
└──────────────────────────────────────────────┘
```

**파일명 파싱 규칙**
- 파일명에서 `(YY.MM)` 패턴 추출 → `20YY년 MM월`로 표시
- 패턴 미매칭 시 파일명 그대로 표시하고 사용자가 월 직접 입력
- 동일 월 파일 재업로드 시 덮어쓰기 확인 후 교체

```
[사용자]
  │
  ├── 연간 사업계획 XLSX 업로드 (연 1회)
```

## 8. 비기능 요구사항

- 파일 업로드 후 **3초 이내** 화면 갱신

- 반응형: 1280px 이상 데스크탑 최적화
